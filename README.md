# Chess-Clock
A simple yet powerful Android Chess Clock App built using Java and Android Studio.
The app provides dual timers for two players, making it ideal for chess matches and other timed games.


 Features

 Dual player timers (White & Black)

 Start, pause, and reset functionality

 Adjustable time limits per player

 Optional sound alerts when time runs out

 Clean, user-friendly interface

 Lightweight and efficient performance



Tech Stack

Language: Java

IDE: Android Studio

Platform: Android

UI: XML Layouts with Material Design




How to Run

1. Clone this repository:

git clone https://github.com/rishikashetty18/Chess-Clock.git


2. Open the project in Android Studio.


3. Let Gradle build the project.


4. Run the app on an emulator or a physical device.
